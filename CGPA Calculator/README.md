# 🧮 CGPA Calculator

A console-based CGPA calculator developed in C++ with text-file handling.

## 📖 Overview

This project was created as part of my Programming Methodology assignment.

The program reads student information from `Student.txt`, displays the raw records in the terminal, calculates student results, and writes the processed information to `Result.txt`.

## ⚙️ Features

* Reads student records from `Student.txt`
* Displays student information in a structured terminal table
* Processes:

  * Assignment 1 marks
  * Assignment 2 marks
  * Test marks
  * Quiz marks
  * Examination marks
* Calculates coursework, examination, overall scores, and grades
* Generates a grade distribution summary
* Displays the total number of students processed
* Writes processed results to `Result.txt`

## 📌 Notes

* The terminal displays the raw student input in a structured format.
* Calculated results and grades are written to `Result.txt`.
* No database is used.
* All information is read from and written to text files.

## 🛠️ Technology

* C++
* File handling

# 🎓 Diploma Assignment Projects

A collection of academic programming projects completed during my Diploma in Information Technology.

These projects were developed for different diploma subjects and helped me gain practical exposure to C++, Java, Android development, file handling, object-oriented programming, and SQLite databases.

---

## 🧮 CGPA Calculator

A console-based CGPA calculator developed in C++ for a Programming Methodology assignment.

The program reads student records from a text file, calculates coursework and overall results, assigns grades, and writes the processed information to an output file.

**Technologies:** C++ and text-file handling

[View Project](./CGPA Calculator)

---

## 📚 Library Management System

A console-based library management system developed in Java for an Object-Oriented Programming assignment.

The system allows users to view available books, borrow and return books, set due dates, and calculate late-return fees.

**Technologies:** Java and object-oriented programming

[View Project](./Library Management System)

---

## 📱 Student Attendance App

An Android application developed for a Mobile Application Development assignment.

The application allows lecturers to manage student records, record daily attendance, and view automatically calculated attendance summaries and percentages.

**Technologies:** Java, Android Studio and SQLite

[View Project](./Student Attendance App)
